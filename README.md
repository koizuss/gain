gain
====

音楽を共感するサービス

今日のパーティーでかける曲を共感しよう
オーディエンスが共感できる曲に「いいね」、DJはオーディエンスが共感した曲を繋ぐ、これでパーティーは大成功！！

## Feauture

- Webアプリケーションとして提供
- Googleアカウントでログイン
- セットリストを登録
- セットリストの中から共感できる曲に「いいね」
- 「いいね」が多い順番でセットリストを表示

## Architecture

- node
- heroku
- GoogleOAuth2
- passportjs
- express
- express-resource
- mongoDB
- mongoose
